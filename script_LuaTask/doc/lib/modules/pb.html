Functions

setStorage (storage, cb)
设置电话本存储区域

Parameters:

  • storage string , 存储区域字符串，仅支持"ME"和"SM"
  • cb , 设置后的回调函数

    回调方式为cb(result)，result为true表示成功，false或者nil表示失败

Returns:

    无

Usage:

    pb.setStorage(storage,cb)
read (index, cb)
读取一条电话本记录

Parameters:

  • index number ，电话本在存储区的位置
  • cb func ，function类型，读取后的回调函数

    回调方式为cb(result,name,number)：result为true表示成功，false或者nil表示失败；name为姓名；number为号码

Usage:

    pb.read(1,cb)
write (index, name, num, cb)
写入一条电话本记录

Parameters:

  • index number ，电话本在存储区的位置
  • name string ，姓名
  • num string ，号码
  • cb func , functionl类型，写入后的回调函数

    回调方式为cb(result)：result为true表示成功，false或者nil表示失败

Returns:

    无

Usage:

    pb.write(1,"zhangsan","13233334444",cb)
delete (index, cb)
删除一条电话本记录

Parameters:

  • index number , 电话本在存储区的位置
  • cb func , function类型，删除后的回调函数

    回调方式为cb(result)：result为true表示成功，false或者nil表示失败

Returns:

    无

Usage:

    pb.delete(1,cb)
