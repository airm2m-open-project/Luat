getState (num) 
查询某个号码的通话状态

  • num 
     查询号码

Usage:

dial (number, delay) 
拨号

Parameters:

  • number 
     号码
  • delay 
     延时delay毫秒后，才发送at命令呼叫，默认不延时

Returns:

  • result true表示允许发送at命令拨号并且发送at，false表示不允许at命令拨号

Usage:

  • cc.dial('10086')
