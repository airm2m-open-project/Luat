anyCallExist ()
是否存在通话

Returns:

    result 存在通话返回true，否则返回false

Usage:

    result = cc.anyCallExist()
getState (num)
查询某个号码的通话状态

Parameters:

  • num 查询号码

Returns:

    state 通话状态，状态值参考本模块定义

Usage:

    state = cc.getState('10086')
dial (number, delay)
拨号

Parameters:

  • number 号码
  • delay 延时delay毫秒后，才发送at命令呼叫，默认不延时

Returns:

    result true表示允许发送at命令拨号并且发送at，false表示不允许at命令拨号

Usage:

    cc.dial('10086')
hangUp (num)
挂断所有通话

Parameters:

  • num 号码，若指定号码通话状态不对 则直接退出 不会执行挂断，若挂断时会挂断所有电话

Returns:

Usage:

    cc.hangUp('10086')
accept (num)
